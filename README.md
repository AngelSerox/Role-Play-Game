# Role-Playing Game (RPG) - freeCodeCamp Project

This is a practice project completed as part of the **freeCodeCamp** curriculum. 
The primary objective of this project was to learn and master foundational programming 
concepts in JavaScript by building a text-based/interactive Role-Playing Game from scratch.

## 🎯 Project Goals
- Understand the fundamentals of JavaScript as a core web technology alongside HTML5 and CSS3.
- Learn how to make web pages interactive by implementing game logic.
- Practice algorithm design and problem-solving through a structured, multi-step application.

## 📋 Core Concepts & Features Implemented
Throughout the development of this RPG, I learned and applied the following programming concepts:
- **Variables & Data Types:** Managing player stats (Health, Gold, XP) and weapons.
- **Control Flow:** Utilizing `if/else` statements and `switch` cases to handle game choices and battle mechanics.
- **Data Structures:** Working with **Arrays** and **Objects** to store complex game data (locations, monsters, inventory).
- **Functions:** Writing reusable blocks of code to handle state changes, attacking, and buying items.
- **Loops:** Iterating through structures to update game elements efficiently.
- **DOM Manipulation:** Dynamically updating the text, buttons, and statistics on the screen using JavaScript.

## 🛠️ Tech Stack
- **JavaScript (ES6+)** (Core game logic and interactivity)
- **HTML5** (Structure of the game interface)
- **CSS3** (Styling and layout)
- **Platform:** freeCodeCamp Interactive Environment
